# Tako Jump - Project Guidelines

## 概要
タコがジャンプで上に登り、月（ゴール）に到達するモバイル向けウェブゲーム

## 技術スタック
- **フレームワーク**: React 18 + TypeScript
- **ビルド**: Vite
- **ゲーム描画**: Canvas API（外部ライブラリ不使用）
- **スタイリング**: Tailwind CSS
- **状態管理**: React hooks + useReducer
- **ストレージ**: localStorage（ハイスコア保存）

## 設計哲学

### アーキテクチャ
- **関心の分離**: ゲームロジック（`/game`）とUI（`/components`）を明確に分離
- **設定の外部化**: 調整可能なパラメータは全て `config.ts` に集約
- **イミュータブル**: ゲーム状態は直接変更せず、新しいオブジェクトを生成

### コード規約
- 関数は単一責任、50行以内を目安
- 型定義は `types.ts` に集約
- マジックナンバー禁止、定数は `config.ts` へ

### モバイルファースト
- タッチイベント優先（mouse fallbackあり）
- 縦画面固定想定（アスペクト比 9:16〜9:20）
- 60fps維持を目標

## ビジュアルスタイル
- **テーマ**: 夜空・宇宙
- **カラーパレット**:
  - 背景: `#2D2A5A`（紺色）
  - 床: `#E8A87C`（サーモンピンク）
  - 水: `#CC00FF`（マゼンタ）
  - 月: `#FFD93D`（黄色）
  - 星: `#9B8AC4`（薄紫）
- **アート**: ピクセルアート風、レトロゲーム感

## 重要な制約
- 二段ジャンプ不可
- 床上での歩行不可（移動はジャンプのみ）
- ジャンプ角度は45°〜135°の範囲
