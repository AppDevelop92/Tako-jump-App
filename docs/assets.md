# アセット仕様

## タコ（主人公）

### 基本サイズ
- 32 x 40 px

### チャージモーション（5段階 + MAX点滅）
| 状態 | 体の色 | 目の状態 | 備考 |
|------|--------|----------|------|
| チャージ0% | 通常色 | 大きく開いている | 待機状態 |
| チャージ25% | やや変化 | やや細まる | |
| チャージ50% | 中間色 | さらに細まる | |
| チャージ75% | 濃い色 | より細まる | |
| チャージ100% | オレンジ⇔黄色 | 完全に細い | 100msごとに高速切り替え（点滅） |

チャージ100%到達時は、オレンジと黄色が交互に高速で切り替わり、最大チャージ状態であることを視覚的に示します。

### 死亡アニメーション（3段階）
| 段階 | 経過時間 | 体の色 | 目の状態 | 説明 |
|------|----------|--------|----------|------|
| 死んだ瞬間 | 0〜300ms | 通常色 | 驚いた表情 | 水に触れた直後 |
| 死んだ直後 | 300〜600ms | 通常色 | バツ印 (X X) | 目がバツに変化 |
| 黒ずみ | 600ms〜 | 黒色 | バツ印 (X X) | 体が黒く変色 |

死亡アニメーションは3段階でなだらかに変化し、キャラクターの死亡を視覚的に表現します。

### 構造
```
    ████████
  ████████████
  ██  ████  ██    ← 目の部分
  ██  ████  ██
  ████████████
    ██  ██  ██    ← 足（3本）
```

- 高さ: 16px 固定

### 見た目

- 1タイル: 16 x 16 px
- 色: `#E8A87C`（サーモンピンク/オレンジ）

## 水

### 見た目
- メインカラー: `#CC00FF`（マゼンタ）
- 上端: 波形（ジグザグ or サイン波）
- 波の高さ: 8px
- 波のアニメーション: 左右に揺れる

## 背景

### ベース
- 色: `#2D2A5A`（紺色）

### 星（小）
- サイズ: 2x2 〜 4x4 px
- 色: `#9B8AC4`（薄紫）
- 種類: ドット、十字（✦）
- 配置: ランダム、密度は画面に20〜30個

### 三日月（装飾）
- サイズ: 12x12 px
- 色: `#7B68A6`（薄紫）
- 配置: 画面に3〜5個

### キラキラ
- サイズ: 8x8 px
- 形: 十字型（✦）
- 色: `#C9B8E0`
- アニメーション: なし（静止）

## UI要素

### フォント
- ピクセルフォント風
- Google Fonts: `Press Start 2P` または `VT323`
- フォールバック: `monospace`

### ライフアイコン
- タコの顔のミニ版
- サイズ: 24x24 px
- 残ライフ: 白、失ったライフ: グレー

### ボタン/パネル
- 背景: `#000000` 不透明度80%
- 枠: `#FFFFFF` 2px
- 角: 直角（ピクセル風）
